The studio

Independent game development

Brumous Studios is a one-person half-assed independent game development studio based in Finland. I'm learning game development on the side while working and trying to finish my studies. Go big or go... brumous?

Currently in development: Renascent, a third-person action RPG set in an original fantasy world rooted in nordic mythology and folklore. Built in Unreal Engine 5.

This is a "for-fun" project, not taking it toooo seriously (do not be fooled by the overkill of a website etc. This is all just me procrastinating on "real" stuff...)
